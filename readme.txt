Building your landing page in Wordpress with drag & drop frontend editing.

== Description ==

Building your landing pages has never been easier with Bloks. Now you can build your landing pages in just seconds with drag & drop from Bloks library with different types of blocks. We have plenty of blocks packed with plugin and continue update the list every week.
No more suffer with html, css, javascript, php messes, your landing page is ready to roll in a blink of eyes.

== [Try Demo](http://demo.bloks.co) ==

== Features ==

- Click to insert blocks to your page and easy to manage
- Full control over each block
- Block color overlay supported
- Background video playing supported
- Slideshow Cover supported
- Portfolio, Tabs supported
- Google Maps supported
- Form building supported
- Custom URL to your landing page
- Multi Color set for each Block
- Support Custom CSS for each Landing page
- Support building your own custom block
- Responsive layout supported
- Frontend editing with tablet, mobile layout preview

== Installation ==

1. Upload plugin to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set your Google API keys for Google Maps to work
4. Start building your Landing Page by click on Bloks >> Add New

Install Bloks plugin as regular Wordpress plugin. No extra steps.

== Frequently Asked Questions ==

= Can I create my own block? =
Yes, you can. You will need a bit knowledge of HTML, CSS and JS. If you have difficulty, please contact us then we will guide you through.

= Can I put my own meta tags, keywords? =
Yes, you can. On each Bloks landing page, you can see the setting icon next to page title, click on that and you will be able to set your own meta tags and keyword description.
